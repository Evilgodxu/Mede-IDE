# 2026-06-22 更新日志：新增开发工具

## 新增功能

### 1. 终端面板
- 新增 `TerminalManager` 和 `TerminalPanel`
- 支持创建多个终端会话
- 实时命令执行和输出显示
- 会话管理（新建、切换、关闭）
- 支持 Android 5.0+ 所有版本（通过反射获取 PID）

### 2. 全局文件搜索
- 新增 `FileSearchEngine` 和 `SearchPanel`
- 支持正则表达式搜索
- 支持大小写敏感/不敏感
- 支持文件名模式过滤（如 `*.kt`）
- 搜索结果高亮显示

### 3. 代码片段模板
- 新增 `CodeSnippet` 和 `SnippetPicker`
- 内置 15+ 种代码模板：
  - Kotlin: main, data class, Composable, suspend, when
  - Java: main, class
  - Python: main, class
  - JavaScript: function, arrow, async
  - Web: HTML5, CSS Flexbox
  - 数据结构: 链表节点, 树节点

### 4. 最近文件列表
- 新增 `RecentFilesManager`
- 自动记录最近打开的文件
- 支持分组显示（今天/昨天/更早）
- 最多保留 50 个文件

### 5. 书签功能
- 新增 `BookmarkManager`
- 支持添加、删除、管理书签
- 书签持久化存储

## 修复的问题

- 修复了 `FileSearchEngine` 中 `emit` 在 `forEach` 高阶函数中的协程推断问题
- 修复了 `TerminalManager` 中 Android API 30 以下 `Process.pid()` 的兼容性问题
- 优化了代码片段模板中的字符串插值问题

## 技术改进

- 终端面板使用 `ProcessBuilder` 执行命令
- 文件搜索使用 `Flow` 实现流式处理，支持大文件搜索
- 所有新功能都支持深色模式
