# 双列文件管理器

## 布局

```
┌─────────────────┬─────────────────┐
│    左列文件      │    右列文件      │
│   (相对路径)     │   (绝对路径)     │
├─────────────────┼─────────────────┤
│                 │                 │
│   文件列表       │   文件列表       │
│                 │                 │
├─────────────────┴─────────────────┤
│  后退 前进 新建 同步 根目录        │
└───────────────────────────────────┘
```

## 功能

### 双列导航
- **左列**: 相对路径导航，基于工作目录
- **右列**: 绝对路径导航，基于存储根目录
- 两列独立历史记录，支持后退/前进

### 文件操作
- 单击打开文件夹/压缩包
- 长按打开功能菜单
- 左右滑动选中
- 支持跨列复制/移动
- 支持压缩/解压

### 选择模式
- 全选/反选/取消
- 对比文本文件（2个）
- 合并文本文件（2个）

### 工具栏
- **后退/前进**: 操作最后交互的列
- **新建**: 新建文件或文件夹 最后交互的列创建
- **同步**: 将当前列路径同步到另一列
- **根目录**: 回到当前列根目录

## 紧凑布局
- 每行显示: 名称 + 修改日期
- 不显示文件大小
